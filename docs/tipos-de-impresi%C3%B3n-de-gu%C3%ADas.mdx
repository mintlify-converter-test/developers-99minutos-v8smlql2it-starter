## Formatos de Impresión

Dentro de [99Minutos](http://99Minutos.com)
 tiene la opción de imprimir dos formatos de guía PDF y ZPL

Estos son

**formato PDF**: Es el archivo mas común y que puedes imprimir usando cualquier impresora tradicional y comercial. (Considera que es en Base64, si usas nuestra API y necesitarás un decodificar o librería en base al lenguaje utilizado)

**formato ZPL:** El lenguaje de programación Zebra (ZPL) es el lenguaje de comandos que utilizan todas las impresoras compatibles con ZPL. Es un lenguaje basado en comandos que utilizan las impresoras como instrucciones para crear las imágenes impresas en las etiquetas.

<Info>

## Cuándo usar ZPL

Este formato es recomendado para impresoras que generan etiquetas autoadheribles, debes de tener muy bien calibrada tu impresora para que todos los campos salgan de manera correcta,

</Info>

Te dejamos los dos tipos de impresora recomendados por [99Minutos](http://99Minutos.com)

*   ZT411
*   ZT230

Te dejamos un breve manual para configurar la impresosas:  
1.- Acceder a la pagina [Impresoras Zebra](https://zadig.akeo.ie/)
  
2.- Descargar programa Zadig 2.7  
3.- Ejecutar programa Zadig 2.7 como administrador  
4.- Seleccionar Options  
5.- Seleccionar opción “List All Devices”  
6.- Seleccionar el dispositivo para instalar o reemplazar el driver y dar clic en el botón “Install Driver“ o “Replace Driver“ y esperar a que finalice la instalación

\## Que tamaños puedo imprimir

La dimensiones de las guías que puedes imprimir desde 99Minutos son:

*   Carta 9 X 11 pulgadas
*   ZPL (Zebra) 6 X 8 Pulgadas
*   Small 4 X 6 pulgadas
