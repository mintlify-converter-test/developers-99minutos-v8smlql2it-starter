title: "Estados de 99minutos"
---
En 99minutos se tienen dos tipos de estados, los estados **Legacy** y los estados **APIV3**.  
**Estados Legacy:** Son estados que se utilizaban en el anterior API de 99minutos.  
**Estados APIV3:** Son estados que se utilizan en la nueva versión del API de 99minutos.  
Ambos estados se siguen utilizando, en la siguiente tabla se muestran los dos tipos de estados, la relación entre ambos y su descripción.
