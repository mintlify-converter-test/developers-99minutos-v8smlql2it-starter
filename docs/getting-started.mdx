## INTRODUCCION

Este manual de uso te brindará un paso a paso detallado para iniciar a integrar 99Minutos en tu negocio, y así poder generar tus Guías para la entrega de tus productos.

<Check>

## Antes de Empezar

Te recomendamos realizar Guías de pruebas para comprobar que 99Minutos esté correctamente integrado con respecto a tus necesidades .

</Check>

## Ambientes

Contamos con dos ambientes en donde convive todo el universo de 99Minutos, para ambos ambientes es necesario que registres y actives tu cuenta, cada cuenta generará su propias credenciales (ClientID y Client Secrect)

| Ambiente | Descripción |
| --- | --- |
| SANDBOX | Ambiente de prueba en donde podrás generar y consultar Guías las cuales no involucran costos ni operación logistica y te permiten comprobar que el proceso de creación de guías funcione de la mejor manera. Además, no necesitas tarjetas ni dinero real.    **Genera tu cuenta:  [https://authsandbox.99minutos.com/](https://authsandbox.99minutos.com/)    Plataforma:**  [https://selfsandbox.99minutos.com/](https://selfsandbox.99minutos.com/)    API URL:\*\*  [https://sandbox.99minutos.com](https://sandbox.99minutos.com) |
| PRODUCTIVO | Ambiente real/ productivo donde los clientes con cuenta activa pueden generar, rastrear, ver o cancelar una o múltiples guías hacia 99minutos.    **Genera tu cuenta en:  [https://auth.99minutos.com/](https://auth.99minutos.com/)    Plataforma:**  [https://envios.99minutos.com/](https://envios.99minutos.com/)    API URL:\*\*  [https://delivery.99minutos.com](https://delivery.99minutos.com) |
